# Android Sentry Demo

A simple demo project used sentry.

Libraries used on the sample project
------------------------------------
* [Sentry](https://github.com/getsentry/sentry-java)
* ViewBinding

Setup
------------------------------------
Set your DSN key in `string.xml` file.
```xml
<resources>
    <string name="sentry_dsn">YOUR_DSN</string>
</resources>
```
